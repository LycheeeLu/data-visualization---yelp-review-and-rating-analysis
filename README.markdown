Goals:
- Extract a subset of Yelp reviews
- Analyze sentiment vs. star rating
- Compare word usage across ratings
- Examine review characteristics (length, usefulness)
- (Optional) Perform basic emotion analysis

If some libraries are missing, install them with `pip install ...` in your environment.


https://business.yelp.com/data/resources/open-dataset/

install the dataset from here and retrieve the `review.json` file and store it into the `data` directory

MacOS: `python3 sentimentStarRating.py ...`
